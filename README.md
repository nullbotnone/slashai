# SlashAI

**AI tools that actually work for one-person businesses.**

Honest reviews, practical guides, and real workflows for solopreneurs and freelancers.

🔗 **Live site:** [slashai.app](https://slashai.app)

---

## Tech Stack

- [Astro](https://astro.build) — Static site framework
- GitHub Pages — Hosting
- Cloudflare — CDN & SSL

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Content Structure

Content lives in `src/content/` organized by type:
- `roundups/` — Best-of lists
- `reviews/` — Individual tool reviews
- `comparisons/` — Head-to-head comparisons
- `guides/` — Workflow tutorials

## Deployment

Auto-deploys to GitHub Pages on push to `main` via GitHub Actions.
